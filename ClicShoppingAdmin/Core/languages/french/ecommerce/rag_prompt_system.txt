text_rag_entity_metadata_guidelines = --- DIRECTIVES SUR LES MÉTADONNÉES D'ENTITÉ ---
Lors du traitement de requêtes qui référencent des entités spécifiques (produits, commandes, clients) :

1. EXTRACTION D'ENTITÉ : Toujours essayer d'extraire entity_id et entity_type depuis :
   - Références directes : "produit 123", "commande #456"
   - Indices contextuels : "ce produit", "ce client"
   - Historique de conversation précédent

2. PROPAGATION D'ENTITÉ : Transmettre les métadonnées d'entité à travers toutes les étapes :
   - Inclure entity_id et entity_type dans les métadonnées d'étape
   - Préserver le contexte d'entité dans les résultats
   - Utiliser le wrapper _entity_metadata si nécessaire

3. TYPES D'ENTITÉ SUPPORTÉS :
   - products (products_id)
   - orders (orders_id)  
   - customers (customers_id)
   - categories (categories_id)

4. FORMAT DES MÉTADONNÉES :
   ```json
   {
     "entity_id": 123,
     "entity_type": "products",
     "_entity_metadata": {
       "entity_id": 123,
       "entity_type": "products"
     }
   }
   ```
