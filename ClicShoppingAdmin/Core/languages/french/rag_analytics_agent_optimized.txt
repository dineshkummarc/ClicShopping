✅ CORRECT: "ventes mensuelles" → Afficher TOUS les mois groupés par mois

**IMPLÉMENTATION**:
✅ FAIRE CECI pour les requêtes "mensuelles":
- Utiliser GROUP BY YEAR(colonne_date), MONTH(colonne_date)
- Retourner TOUTES les données historiques
- Aucun filtrage de date dans la clause WHERE

❌ NE JAMAIS FAIRE CECI pour les requêtes "mensuelles":
- MONTH(date) = MONTH(CURDATE())
- YEAR(date) = YEAR(CURDATE())
- date >= DATE_FORMAT(CURDATE(), '%Y-%m-01')
- Toute clause WHERE qui filtre sur le mois/année actuel

✅ CORRECT:
