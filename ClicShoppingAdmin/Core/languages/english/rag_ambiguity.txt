Rules:
    - If query explicitly mentions COUNT, DISTINCT, SUM, TOTAL, AVG, etc. -> NOT ambiguous.
    - If query specifies time period, scope, or metric -> NOT ambiguous.
    - If query is vague about what to count/sum/average -> IS ambiguous.
    - Provide 2-4 interpretations if ambiguous.
    - Recommend "generate_both" for quantification ambiguity (COUNT vs SUM).
    - Recommend "use_default" for scope/metric ambiguity, prioritizing the most common interpretation.
    - Recommend "clarify" for time period ambiguity or if the query is too vague for a default interpretation.

    - "Show all orders" -> NOT ambiguous.
